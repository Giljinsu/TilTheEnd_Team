# 주요 코드

### 세션 얻기

```
String ID = (String) httpSession.getAttribute("userId");
```

getAttribute를 통해 유저의 id를 가져온다.

### 구름

```
<input class="form-control" type="text" name="id" value="<%=id%>" id="id"/>
```

구름 IDE에서는 input 태그의 id가 대문자면 인식하지 못한다.
